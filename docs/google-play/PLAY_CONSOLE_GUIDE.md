# Guia paso a paso Play Console

## Paso 1 - Cuenta de desarrollador

1. Crear/verificar cuenta.
2. Completar identidad y contacto.
3. Aceptar acuerdos.
4. Pagar tasa si corresponde.

## Paso 2 - Crear aplicacion

1. Idioma predeterminado.
2. Nombre.
3. Tipo: App (no juego).
4. Gratis o pago.
5. Correo de contacto.
6. Activar Play App Signing.

## Paso 3 - Ficha de tienda

Subir:

- titulo;
- descripcion corta;
- descripcion completa;
- icono;
- feature graphic;
- capturas;
- categoria y etiquetas;
- email/web;
- URL politica de privacidad.

## Paso 4 - Contenido de aplicacion

Completar:

- Politica de privacidad.
- Anuncios.
- App access.
- Publico objetivo.
- Clasificacion de contenido.
- Data Safety.
- Permisos.

## Paso 5 - Paises y precio

1. Elegir paises.
2. Definir gratuita/pago.
3. Revisar implicaciones de app gratuita.

## Paso 6 - Subir AAB

1. Crear release de test interno.
2. Subir `.aab`.
3. Pegar release notes.
4. Revisar warnings/errores.

## Paso 7 - Pruebas

1. Ejecutar test interno.
2. Ejecutar test cerrado si aplica.
3. Corregir incidencias.

## Paso 8 - Solicitar produccion

1. Completar cuestionario de preparacion.
2. Resumir pruebas y feedback.
3. Solicitar acceso.

## Paso 9 - Publicacion

1. Crear release de produccion.
2. Revisar summary.
3. Publicacion gestionada o inmediata.
4. Enviar a revision.

## Paso 10 - Post-publicacion

1. Monitorizar Android Vitals.
2. Revisar crashes/ANR.
3. Gestionar feedback de usuarios.
4. Incrementar `versionCode` para siguiente release.

## Nota sobre requisito de test cerrado

Algunas cuentas personales nuevas requieren test cerrado con participantes por periodo minimo antes de solicitar produccion. Verificar este requisito directamente en Play Console de la cuenta del propietario.
