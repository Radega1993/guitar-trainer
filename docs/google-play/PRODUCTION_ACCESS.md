# Production access (Google Play)

## Estado regulatorio

Segun tipo y fecha de la cuenta de desarrollador, Google puede requerir:

- test interno/cerrado previo;
- para algunas cuentas personales nuevas, test cerrado con participantes durante periodo minimo.

## Accion requerida

El propietario debe verificar en Play Console si la cuenta esta sujeta al requisito de test cerrado previo a produccion.

## Recomendacion

1. Publicar primero en test interno.
2. Pasar a test cerrado si aplica.
3. Solicitar acceso a produccion con evidencia de pruebas.
